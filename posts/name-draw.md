A Secret Santa name draw tool with seasonal themes, blocked pairs, and a hat shuffle animation.

- Draws names using the Sattolo cycle (guaranteed no self-assignments)
- Block specific pairs from being matched
- 12 seasonal themes auto-applied by month (or pick from the dropdown)
- Names and settings persist in localStorage
