# gobloom.io

Personal site and blog. Built with Next.js — static rendering, markdown posts, animated SVG previews.

## About

I'm Daniel — Founding Engineer at [Plantiful](https://tryplantiful.com), formerly CTO at [Private Prep](https://privateprep.com) ($15MM+ ARR). I write about frontend patterns, React, and engineering in general.

The blog skews toward things I've actually shipped: hooks, CSS micro-interactions, AI tooling, developer utilities.

## Stack

- Next.js 15 · React 19
- Markdown (gray-matter) for posts
- Custom animated SVGs as post previews
- Static export, deployed to Vercel

## Running locally

```bash
pnpm dev
```
